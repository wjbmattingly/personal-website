This project is a wrapper for integrating GLiNER, a named entity recognition (NER) model, with the spaCy natural language processing library. GLiNER, which stands for Generalized Language INdependent Entity Recognition, is an advanced model for recognizing entities in text. The spaCy wrapper enables easy integration and use of GLiNER within the spaCy environment, enhancing NER capabilities with GLiNER's advanced features.
